## Описание программы

Currency Converter - это графическое приложение для конвертации валют с использованием актуальных курсов из внешнего API. Программа позволяет:

- **Конвертировать суммы** между различными валютами в реальном времени
- **Получать актуальные курсы** из API exchangerate-api.com
- **Сохранять историю операций** в JSON файл и загружать её
- **Просматривать историю** всех выполненных конвертаций
- **Проверять корректность** вводимых данных

Приложение имеет интуитивно понятный интерфейс на русском языке и подходит для повседневного использования.

## Получение API ключа

Для работы приложения необходимо получить бесплатный API ключ от ExchangeRate API:

### Инструкция по получению API ключа:

1. Перейдите на сайт: https://www.exchangerate-api.com/
2. Нажмите кнопку **"Get Free Key"**
3. Введите ваш email адрес
4. Подтвердите регистрацию, перейдя по ссылке из письма
5. После подтверждения вы получите API ключ (бесплатный тариф позволяет 1500 запросов в месяц)
6. Скопируйте полученный ключ

### Настройка API ключа в программе:

1. Откройте файл `currency_converter.py`
2. Найдите строку:
   ```python
   self.api_key = "YOUR_API_KEY_HERE"

# Использование программы 
1. Выберите исходную валюту из выпадающего списка ("Из валюты")
2. Введите сумму для конвертации (только положительные числа)
3. Выберите целевую валюту ("В валюту")
4. Нажмите "Конвертировать"
5. Результат отобразится в поле "Результат" и будет добавлен в историю

# Дополнительные функции
Обновить курсы:

* Показывает актуальный курс между выбранными валютами
* Обновляет информацию в реальном времени

Очистить поля:
* Очищает поле ввода суммы и сбрасывает результат

Управление историей:

* Сохранить историю - сохраняет все операции в JSON файл
* Загрузить историю - загружает ранее сохраненную историю
* Очистить историю - удаляет все записи из текущей истории
